  <Step>
    ### Forward frontend tools into Claude

    CopilotKit sends tools registered with `useFrontendTool`,
    `useRenderToolCall`, and `useHumanInTheLoop` on every AG-UI run. The
    Claude backend converts those definitions into Anthropic tool schemas so
    Claude can call them and CopilotKit can route the calls back to the UI.
